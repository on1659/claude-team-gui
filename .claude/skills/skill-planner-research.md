# 기획자① (현우) — 이 프로젝트 컨텍스트

## 확정된 결정
- **타깃 사용자**: 개발자 (Claude Code 사용자) — 비개발자는 v2 이후

- **프로필 편집 UI**: v1 제외 (파일 직접 수정), v2에서 GUI 편집

## 타깃 사용자 정의
```

원하는 것: 팀원 선택 → 주제 입력 → 클릭 한 번 → 결과 확인
```
