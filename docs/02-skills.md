# 스킬 4종 — 비교 및 토큰 비용

## 스킬 개요

| 스킬 | 구조 | 인원 | 용도 |
|------|------|------|------|
| `/meeting` | Claude 1개, 순차 | 소수 | 빠른 브레인스토밍 |
| `/meeting-multi` | Claude 1개, 순차, 누적 | 기본 8명 | 자연스러운 토론 흐름 |
| `/meeting-agent` | Claude 여러 개, 병렬 | 기본 8명 | 편향 없는 독립 분석 |
| `/meeting-team` | Claude 여러 개, 병렬 | 기본 8명 | 팀원 정체성 기반 분석 |

---

## 스킬 상세

### `/meeting` — 경량 회의
- bkit 내장 스킬
- Claude 1개, 소수 역할 순차 실행
- 간단한 검토에 적합

### `/meeting-multi` — 역할극 회의 (8인)
- 파일: `.claude/commands/meeting-multi.md`
- Claude 1개가 8개 역할 순서대로 연기
- 앞 발언에 영향받는 구조 → 자연스러운 토론 흐름
- 한계: 편향 가능, 인원 많을수록 토큰 폭증

### `/meeting-agent` — 서브에이전트 병렬 회의
- 파일: `.claude/commands/meeting-agent.md`
- 역할별 독립 서브에이전트 병렬 실행
- 서로 모르는 상태에서 독립 분석 → 편향 없음
- 역할 기반 (직군) — 사람이 아닌 기능

### `/meeting-team` — 팀원 프로필 기반 회의
- 파일: `.claude/commands/meeting-team.md`
- 프로필 DB: `.claude/meeting-team-profiles.md`
- 팀원 이름, 성격, 말투, 판단 기준을 주입한 서브에이전트
- 가장 전문적이고 일관된 분석

## 토큰 비교 (8인 기준)

### 절대값

| 스킬 | input | output | 총합 |
|------|-------|--------|------|
| `/meeting` | ~2,000 | ~900 | **~2,900** |
| `/meeting-agent` | ~8,700 | ~2,900 | **~11,600** |
| `/meeting-team` | ~14,300 | ~3,800 | **~18,100** |
| `/meeting-multi` | ~16,300 | ~2,900 | **~19,200** |

### `/meeting` 대비 배율

| 스킬 | 배율 |
|------|------|
| `/meeting` | **1x** |
| `/meeting-agent` | **4x** |
| `/meeting-team` | **6.2x** |
| `/meeting-multi` | **6.6x** |

### 인원별 효율 (역할극 vs 서브에이전트)

| 인원 | `/meeting-multi` | `/meeting-agent` |
|------|-----------------|-----------------|
| 2명 | ~3,900 | ~3,800 |
| 4명 | ~8,100 | ~6,000 |
| 8명 | ~19,200 | ~11,600 |

> **핵심**: 역할극은 인원 증가 시 제곱에 가깝게 폭증. 서브에이전트는 선형 증가.
> 8명 기준 서브에이전트가 역할극보다 **40% 저렴**.

### 의외의 결과

역할극(multi)이 서브에이전트(agent)보다 비싸다. 8명이 순차 실행되면서 컨텍스트가 누적되기 때문.

## 언제 어떤 스킬을?

| 상황 | 추천 |
|------|------|
| 빠른 검토 | `/meeting` |
| 자연스러운 토론 흐름 | `/meeting-multi` |
| 편향 없는 독립 의견 | `/meeting-agent` |
| 인원 5명 이상 | `/meeting-agent` (토큰 효율) |
| 일관된 팀원 캐릭터 | `/meeting-team` |
| 중요한 아키텍처 결정 | `/meeting-team` |
