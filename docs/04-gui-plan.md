# GUI 계획 — 팀원 고용 시스템

## 개요

CLI 스킬로 검증한 개념을 GUI 앱으로 시각화.
팀원 고용/해고, 회의 실행, 비용 예측을 직관적 인터페이스로 제공.

---

## 화면 구성 (초안)

```
┌─────────────────────────────────────┐
│  인사팀                               │
├────────────────┬────────────────────┤
│  정규직 (글로벌) │  비정규직 (프로젝트) │
│                │                    │
│  지민  (PD)    │  [이 프로젝트 전용]  │
│  현우  (기획)  │                    │
│  태준  (BE)    │  + 계약직 채용      │
│  ...           │                    │
│  + 정규직 채용  │                    │
├─────────────────────────────────────┤
│  오늘의 프리랜서                       │
│  [회의 시작 시 자동 고용 → 종료 후 소멸] │
├─────────────────────────────────────┤
│  예상 인건비: ~11,600 토큰 (4x)       │
│  회의 방식: [multi] [agent] [team]   │
└─────────────────────────────────────┘
```

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| 팀원 카드 | 이름, 직군, 성격 요약 표시 |
| 고용/해고 | 클릭으로 팀 구성 |
| 발령 | 글로벌 ↔ 프로젝트 이동 |
| 예상 비용 | 선택 인원 × 방식 → 토큰 예측 |
| 회의 실행 | 방식 선택 후 주제 입력 → 실행 |
| 팀원 추가 | 폼으로 profiles.md에 자동 기록 |

---

## 개발 순서

1. **CLI 스킬 검증 완료** ← 현재 위치
2. GUI 프레임워크 결정 (웹앱 / Electron / Tauri)
3. 팀원 DB 구조 확정 (profiles.md → JSON 또는 유지)
4. UI 목업 설계
5. 구현

---

## 기술 고려사항

- **글로벌 vs 프로젝트 종속** 선택 가능하게 설계
- profiles.md는 글로벌 공유, 프로젝트별 팀 구성은 별도 파일
- 토큰 예측 계산기 내장
- 프레임워크 미결정 — 웹앱 / Electron / Tauri 중 선택 필요
